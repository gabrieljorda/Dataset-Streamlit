📊 Dataset App – Análise de Dados com Pandas, Plotly e Streamlit

Este repositório tem como objetivo criar, manipular e visualizar um dataset utilizando Python, com foco em estudos de análise de dados e desenvolvimento de aplicações interativas.

O projeto foi desenvolvido utilizando:

-🐼 Pandas – Criação e manipulação do dataset

-📈 Plotly – Visualizações interativas

-🚀 Streamlit – Construção da aplicação web e deploy

-🔎 Sobre o Projeto

O projeto consiste na:

-Criação de um dataset utilizando Pandas

-Tratamento e organização dos dados

-Geração de análises exploratórias

-Construção de gráficos interativos com Plotly

-Desenvolvimento de uma aplicação web com Streamlit

-Deploy da aplicação utilizando o próprio Streamlit Cloud

A aplicação foi desenvolvida com foco em aprendizado prático de:

-Estruturação de dados

-Análise exploratória (EDA)

-Visualização interativa

-Criação de dashboards

-Deploy de aplicações de dados
